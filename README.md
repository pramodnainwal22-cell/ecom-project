# 🛒 E-Commerce Project

This is a full-stack eCommerce project built using React.js for frontend and JSON Server for backend.

---

## 🚀 Features

- User-friendly UI
- Product listing
- Category & Subcategory filtering
- Brand-wise products
- Backend using JSON Server
- Dynamic data rendering

---

## 🛠️ Tech Stack

- Frontend: React.js, Vite
- Backend: JSON Server
- Styling: Bootstrap / CSS

---

## 📁 Project Structure

ecom-project/
│── src/
│── public/
│── server/
│   ├── public/
│   └── data.json
│── package.json

---

## ⚙️ Run Locally

### 1. Clone the repository

git clone https://github.com/pramodnainwal22-cell/ecom-project.git

### 2. Install dependencies

npm install

### 3. Run frontend

npm run dev

### 4. Run backend (JSON Server)

cd server  
npx json-server data.json --watch --port 8000
